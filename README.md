1. **Clone the repository**
   ```bash
   git clone https://github.com/brasdefer1597-cloud/chalamandra-core.git
   cd chalamandra-core
```

1. Load in Chrome
   · Open chrome://extensions/
   · Enable "Developer mode"
   · Click "Load unpacked"
   · Select the chalamandra-core folder
